# **App Name**: New SIMANJA

## Core Features:

- Asset Listing: Display a list of all assets owned by the team.
- Asset Details: Show detailed information for each asset, including purchase date, cost, and current condition.
- Asset Tracking: Allow tracking the location and status of each asset.
- Asset Assignment: Enable assigning assets to specific team members.

## Style Guidelines:

- Primary color: Neutral light gray (#F0F0F0) to provide a clean background.
- Background color: White (#FFFFFF) for a simple design.
- Accent color: Dark gray (#333333) for text and interactive elements.
- Body and headline font: 'Inter' sans-serif font to provide a clean user experience.
- Simple, outline-style icons for asset categories and actions.
- Clean, whitespace-rich layout with clear sections for each asset.
- Subtle transitions and animations for a smooth user experience.
